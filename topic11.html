Viewer features

Crystal Reports provides features for data analysis and presentation. Ideally, these features should be preserved when a report is viewed over the Web. It is therefore important that you select the appropriate report viewer for your Web applications, so that reports are displayed with their design features intact. xxxxxxx
Before deciding which report viewer to use in your web application, you need to consider the reporting features provided by each one, and be aware of any limitations that might be imposed by their use.
General viewer capabilities
This table describes the general capabilities of each viewer:
Viewer
Description
DHTML report page viewer
Provides basic thin-client report viewing capabilities.
report parts viewer
Provides the capability to view individual report parts, including charts, text, and fields.
